# Dashboard de Performance de Vendas (Power BI)

## Visão Geral do Projeto

Este projeto foi desenvolvido com o objetivo de praticar análise de dados e criação de dashboards utilizando Power BI.

O dashboard permite analisar a performance de vendas através de diferentes dimensões do negócio, como região, produto, vendedor e evolução temporal das vendas.

---

## Ferramentas Utilizadas

- Power BI Desktop
- Power Query
- DAX

---

## Dataset

O dataset contém dados simulados de vendas com as seguintes informações:

- Data da venda
- Produto
- Categoria
- Região
- Vendedor
- Quantidade vendida
- Preço unitário

---

## Indicadores analisados

- Faturamento Total
- Quantidade Total Vendida
- Número de Vendas
- Ticket Médio
